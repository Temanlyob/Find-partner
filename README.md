# VibeSync

VibeSync is a modern vibe-based matching and dating platform where users answer personality questions to discover highly compatible people.

The platform focuses on:
- personality matching
- emotional compatibility
- communication style
- vibe connection
- relationship energy

Instead of traditional swipe-only systems, VibeSync uses deep question-based matching to create meaningful connections.

---

# Features

- Multi-language support (English / Hindi)
- 100 personality questions
- Vibe compatibility matching
- Profile system
- Match percentage system
- Save draft support
- Dark modern UI
- Chat section
- Mobile responsive design

# Matching System

Users answer personality and vibe-related questions.

The system analyzes:
- emotional nature
- social behavior
- humor style
- communication vibe
- loyalty
- energy level

Then compatible users are matched using vibe tags and compatibility scoring.
